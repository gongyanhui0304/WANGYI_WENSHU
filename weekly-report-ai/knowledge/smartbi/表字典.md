# SmartBI 表字典

> 信息中心周报所涉及的全部数据表，按系统分类。共 **15 张表**，涵盖 SAP、BI、PLM、AI 四个系统。

---

## 一、SAP 系统

### 1. ODS_SAP_ZBIT_LOG_BM — SAP 部门活跃度汇总

| 属性     | 说明                                   |
| -------- | -------------------------------------- |
| 库       | `express_ods`                        |
| 用途     | 各部门 SAP 账户数 + 登录次数（月汇总） |
| 关联 SQL | 001, 002, 003                          |

| 字段          | 类型    | 说明                 |
| ------------- | ------- | -------------------- |
| KOSTL         | varchar | 部门（成本中心名称） |
| SAL_DATE      | varchar | 月份，取前6位=YYYYMM |
| USERNUMBER_KY | int     | 部门可用账户数       |
| LOGNUMBER     | int     | 部门登录次数         |

---

### 2. ODS_SAP_ZVS_TCODE_LOG — SAP TCODE 访问日志

| 属性     | 说明                         |
| -------- | ---------------------------- |
| 库       | `express_ods`              |
| 数据量   | ~64万行（2025-08~2026-07）  |
| 用途     | 每个 TCODE 的每次访问记录    |
| 关联 SQL | 004, 005, 006, 009, 014, 019 |

| 字段      | 类型        | 说明                          |
| --------- | ----------- | ----------------------------- |
| SLGUSER   | varchar     | 操作用户ID                    |
| NAME_TEXT | varchar     | 用户姓名                      |
| SLGTC     | varchar     | TCODE（事务代码）             |
| TTEXT     | varchar     | TCODE 描述（与 TSTCT 表一致） |
| SLGDATTIM | varchar(14) | 操作时间，YYYYMMDDHHMMSS      |
| TYPE      | varchar     | 模块分类                      |
| KOSTL     | varchar     | 成本中心                      |
| SLGREPNA  | varchar     | 关联程序名                    |

## 二、BI 系统

### 8. DWD_BI_DEPMENT_REPORT — BI 部门活跃度（DW视图）

| 属性     | 说明                              |
| -------- | --------------------------------- |
| 库       | `express_dw`                    |
| 用途     | BI 各部门活跃度（预聚合，直接查） |
| 关联 SQL | 007, 012, 017                     |

| 字段       | 类型     | 说明                    |
| ---------- | -------- | ----------------------- |
| 部门       | nvarchar | 部门名称                |
| 年份月份   | varchar  | 月份 YYYYMM             |
| 各部门总数 | int      | 部门账户数              |
| 总数       | int      | 全公司账户数            |
| 天数       | int      | 当月工作天数            |
| 登录次数   | int      | 部门登录次数            |
| 账户占比   | float    | 部门账户/总账户         |
| 访问率     | float    | 登录次数/(账户数×天数) |
