## 一、SAP 系统指标

### 指标1：各部门账户数
| 属性 | 说明 |
|------|------|
| 含义 | 每个成本中心（部门）在 SAP 中的可用账户数量 |
| 公式 | `SUM(USERNUMBER_KY) GROUP BY KOSTL` |
| 数据源 | `ODS_SAP_ZBIT_LOG_BM` |
| 维度 | 部门 × 月份 |
| SQL | 001 |

### 指标2：部门账户占比
| 属性 | 说明 |
|------|------|
| 含义 | 该部门账户数占全公司总账户数的比例 |
| 公式 | `部门账户数 / 全公司总账户数` |
| 数据源 | `ODS_SAP_ZBIT_LOG_BM` |
| SQL | 001 |

### 指标3：部门登录活跃度（访问率）
| 属性 | 说明 |
|------|------|
| 含义 | 该部门实际登录次数占应登录次数的比例 |
| 公式 | `登录次数 / (账户数 × 工作天数)` |
| 数据源 | `ODS_SAP_ZBIT_LOG_BM` + `DIM_DATE` |
| SQL | 001 |

### 指标4：报表访问次数（TCODE 维度）
| 属性 | 说明 |
|------|------|
| 含义 | 每个 SAP TCODE 在统计周期内的总访问次数 |
| 公式 | `COUNT(SLGUSER) GROUP BY SLGTC` |
| 数据源 | `ODS_SAP_ZVS_TCODE_LOG` |
| 维度 | TCODE × 月 |
| SQL | 004, 009 |

### 指标5：报表访问占比
| 属性 | 说明 |
|------|------|
| 含义 | 该 TCODE 访问次数占所有 TCODE 总访问次数的比例 |
| 公式 | `TCODE访问次数 / 全月总访问次数` |
| 数据源 | `ODS_SAP_ZVS_TCODE_LOG` |
| SQL | 004 |

### 指标6：SAP 模块使用率
| 属性 | 说明 |
|------|------|
| 含义 | 每个 SAP 模块（成本核算/财务管理/报价平台等）的实际使用率 |
| 公式 | `用户每月累计访问次数 / (每个模块应访问次数 × 工作天数)` |
| 数据源 | `ODS_SAP_ZVS_TCODE_LOG` + `ODS_SAP_ZVS_USER_TCODE` + `DIM_DATE` |
| 维度 | 模块 × 月 |
| SQL | 019 |

### 指标7：SAP 各模块使用占比
| 属性 | 说明 |
|------|------|
| 含义 | 各模块（成本核算/财务管理/报价平台/产供销/其他）的使用占比 |
| 公式 | 直接从 `ODS_SAP_ZTBASIS02_02` 读取，A1PERC~A5PERC / 100 |
| 数据源 | `ODS_SAP_ZTBASIS02_02` |
| SQL | 010 |

## 二、BI 系统指标

### 指标8：BI 部门活跃度
| 属性 | 说明 |
|------|------|
| 含义 | 各部门在 BI 系统中的账户数、登录次数、访问率 |
| 公式 | 直接从 DW 视图读取（已预聚合） |
| 数据源 | `DWD_BI_DEPMENT_REPORT` |
| 维度 | 部门 × 月 |
| SQL | 007 |

### 指标9：BI 账户占比
| 属性 | 说明 |
|------|------|
| 含义 | 各部门 BI 账户数占全公司的比例 |
| 公式 | `各部门总数 / 总数` |
| 数据源 | `DWD_BI_DEPMENT_REPORT` |
| SQL | 007 |

### 指标10：BI 访问率
| 属性 | 说明 |
|------|------|
| 含义 | 部门 BI 登录次数 / (账户数 × 天数) |
| 公式 | `登录次数 / (各部门总数 × 天数)` |
| 数据源 | `DWD_BI_DEPMENT_REPORT` |
| SQL | 007 |

### 指标11：BI 报表 TOP20
| 属性 | 说明 |
|------|------|
| 含义 | BI 系统中访问次数最高的 20 个报表 |
| 公式 | 直接从 DW 视图读取 |
| 数据源 | `DWD_BI_TOP20_REPORT` |
| SQL | 011 |

### 指标12：BI 报表占比
| 属性 | 说明 |
|------|------|
| 含义 | 该报表访问次数占全月总访问次数的比例 |
| 公式 | `访问次数 / 总次数` |
| 数据源 | `DWD_BI_TOP20_REPORT` |
| SQL | 011 |

## 三、PLM 系统指标

### 指标13：PLM 部门登录次数
| 属性 | 说明 |
|------|------|
| 含义 | 各部门在 PLM 系统中的登录次数 |
| 公式 | `COUNT(PLMUSER) GROUP BY deptname2` |
| 数据源 | `ODS_PLM_PLMLOGGER` + `ODS_EHR_VPS_EMPINFO` |
| 维度 | 部门 × 月 |
| SQL | 008 |

## 四、AI 系统指标

### 指标14：AI 日访问次数
| 属性 | 说明 |
|------|------|
| 含义 | AI 助手每天的对话消息数 |
| 公式 | `COUNT(id) GROUP BY 日期` |
| 数据源 | `ODS_AI_messages_export` |
| 维度 | 日 |
| SQL | 013 |

### 指标15：AI 周访问次数
| 属性 | 说明 |
|------|------|
| 含义 | AI 助手每周的对话消息数 |
| 公式 | `COUNT(id) GROUP BY 年周` |
| 数据源 | `ODS_AI_messages_export` |
| 维度 | 年+周 |
| SQL | 018 |

## 五、指标-表血缘

```
指标1~3     ← ODS_SAP_ZBIT_LOG_BM + DIM_DATE
指标4~5     ← ODS_SAP_ZVS_TCODE_LOG
指标6       ← ODS_SAP_ZVS_TCODE_LOG + ODS_SAP_ZVS_USER_TCODE + DIM_DATE
指标7       ← ODS_SAP_ZTBASIS02_02
指标8~10    ← DWD_BI_DEPMENT_REPORT (DW视图)
指标11~12   ← DWD_BI_TOP20_REPORT (DW视图)
指标13      ← ODS_PLM_PLMLOGGER + ODS_EHR_VPS_EMPINFO
指标14~15   ← ODS_AI_messages_export
```
