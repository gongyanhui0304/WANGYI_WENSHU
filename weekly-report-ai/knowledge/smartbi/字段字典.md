## 五、比率相关（计算字段）

| 字段名 | 业务含义 | 公式 | 所在SQL |
|--------|----------|------|---------|
| 账户占比 | 部门账户/总账户 | `ROUND(各部门总数/总数, 2)` | 001, 007 |
| 访问率 | 实际登录/应登录 | `ROUND(登录次数/(账户数×天数), 2)` | 001, 007 |
| 占比 | 报表访问/总访问 | `ROUND(访问次数/总次数, 4)` | 004, 011 |
| 模块使用率 | 模块实际使用率 | `累计访问/(应访问×天数)` | 019 |

## 六、参数变量

SQL 模板中使用 `${...}` 作为参数占位符，运行时由 SmartBI 仪表板或 Python 代码替换：

| 参数名 | 类型 | 示例值 | 说明 |
|--------|------|--------|------|
| `${年月}` | varchar(6) | `'202606'` | 查询月份 YYYYMM |
| `${年周}` | varchar | `'2026年25周'` | 查询周次 |

## 七、常见过滤条件

| 条件 | 含义 | 出现SQL |
|------|------|---------|
| `KOSTL <> '信息中心'` | 排除信息中心本身 | 001 |
| `部门 <> '信息中心'` | 排除信息中心 | 007 |
| `SLGTC <> 'SEU_INT'` | 排除系统内部TCODE | 004, 009, 019 |
| `LEFT(SLGTC,6) <> 'ZTBASI'` | 排除 BASIS 类 TCODE | 004, 009 |
| `TCODE LIKE 'Z%'` | 只看自开发 TCODE | 004, 009, 019 |
| `TCODE NOT LIKE 'ZMOM%'` | 排除 MOM 系统 TCODE | 009 |
| `USER_CODE LIKE '0%'` | 只看正式员工 | 001(旧版) |
| `软件开发部` → `可诺特软件` | 部门名称映射 | 007, 008 |
