### 周报关联字段

| 周报用途 | 使用字段 | 指标 |
|----------|----------|------|
| 需求总数 | id COUNT | M201 |
| 本周新增 | created_on ∈ [周一,周日) | M202 |
| 本周完成 | closed_on ∈ [周一,周日) | M203 |
| 本周更新 | updated_on ∈ [周一,周日) | M604 |
| 延期需求 | due_date < TODAY + is_closed=false | M204/M605 |
| 进行中 | is_closed = false | M006 |
| 已完成 | is_closed = true | M007 |
| 优先级分布 | priority_id → enumerations.name | — |
| 完成率 | 已完成/总数 | M606 |

## 四、issue_statuses 表（6个字段，6行）

| # | 字段名 | 数据类型 | 约束 | 键 | 说明 |
|---|--------|----------|------|-----|------|
| 1 | id | integer | NOT NULL | PK | 状态ID |
| 2 | name | varchar(30) | NOT NULL | — | 状态名称 |
| 3 | is_closed | boolean | NOT NULL | — | **是否关闭(周报核心判定)** |
| 4 | position | integer | NULL | — | 排序 |
| 5 | default_done_ratio | integer | NULL | — | 默认进度% |
| 6 | description | varchar | NULL | — | 描述 |

| ID | name | is_closed | 实际含义 |
|----|------|-----------|----------|
| 1 | 新建 | false | 待分配/待开始 |
| 2 | 进行中 | false | 正在处理 |
| 3 | 已解决 | true | 已修复/已实现 |
| 4 | 待验证 | true | 等待测试确认 |
| 5 | 已关闭 | true | 已上线/已验收 |
| 6 | 已拒绝 | true | 不处理/重复 |
