## 一、需求总量指标

### M201 — Issue 总数

| 属性 | 说明 |
|------|------|
| 含义 | Redmine 系统中全部 Issue 数量 |
| 公式 | `SELECT COUNT(*) FROM issues` |
| 数据源 | issues 表 |
| 本期值 | 1,757 |
| SQL | `knowledge/redmine/sql/metrics/M201_total_issues.sql` |

### M006 — 进行中需求数

| 属性 | 说明 |
|------|------|
| 含义 | 状态未关闭的需求 |
| 公式 | `COUNT(*) WHERE is_closed = FALSE` |
| 本期值 | 423 (24.1%) |

### M007 — 已完成需求数

| 属性 | 说明 |
|------|------|
| 含义 | 状态已关闭的需求 |
| 公式 | `COUNT(*) WHERE is_closed = TRUE` |
| 本期值 | 1,334 (75.9%) |

## 二、本周流转指标

### M202 — 本周新增需求

| 属性 | 说明 |
|------|------|
| 含义 | 本周一至周日创建的 Issue |
| 公式 | `COUNT(*) WHERE created_on >= :monday AND created_on < :next_monday` |
| 数据源 | issues.created_on |
| 本周值 | 375 |
| 上周值 | 552 |
| 环比 | ↓ -32.1% |

### M203 — 本周完成需求

| 属性 | 说明 |
|------|------|
| 含义 | 本周一至周日关闭的 Issue |
| 公式 | `COUNT(*) WHERE closed_on >= :monday AND closed_on < :next_monday` |
| 数据源 | issues.closed_on |
| 本周值 | 382 |
| 上周值 | 445 |
| 环比 | ↓ -14.2% |

### M604 — 本周更新需求

| 属性 | 说明 |
|------|------|
| 含义 | 本周一至周日有更新的 Issue |
| 公式 | `COUNT(*) WHERE updated_on >= :monday AND updated_on < :next_monday` |
| 数据源 | issues.updated_on |
| 本周值 | 785 |
| 上周值 | 474 |
| 环比 | ↑ +65.6% |

### M606 — 需求完成率

| 属性 | 说明 |
|------|------|
| 含义 | 已完成需求占总需求的比例 |
| 公式 | `M007 / M201 × 100` |
| 本期值 | 75.9% |
