## 一、核心业务表（周报重点关注）

### 1. issues — 需求/问题主表

| 属性 | 说明 |
|------|------|
| 行数 | 1,757 |
| 用途 | Redmine 核心表，记录所有需求、Bug、任务 |
| 关联表 | projects, users, issue_statuses, enumerations, trackers, issue_categories, versions |

| 字段 | 类型 | 说明 |
|------|------|------|
| id | integer NOT NULL | 主键，Issue编号 |
| tracker_id | integer NOT NULL | 需求类型 FK→trackers |
| project_id | integer NOT NULL | 所属项目 FK→projects |
| subject | varchar NOT NULL | 主题/标题 |
| description | text | 详细描述 |
| due_date | date | 到期日 |
| category_id | integer | 分类 FK→issue_categories |
| status_id | integer NOT NULL | 状态 FK→issue_statuses |
| assigned_to_id | integer | 负责人 FK→users |
| priority_id | integer NOT NULL | 优先级 FK→enumerations |
| fixed_version_id | integer | 目标版本 FK→versions |
| author_id | integer NOT NULL | 创建人 FK→users |
| created_on | timestamp | 创建时间 |
| updated_on | timestamp | 最后更新时间 |
| start_date | date | 开始日期 |
| done_ratio | integer NOT NULL | 进度百分比(0-100) |
| estimated_hours | double precision | 预估工时 |
| parent_id | integer | 父Issue FK→issues |
| root_id | integer | 根Issue |
| lft / rgt | integer | 嵌套树左右值 |
| is_private | boolean NOT NULL | 是否私有 |
| closed_on | timestamp | 关闭时间 |
| lock_version | integer NOT NULL | 乐观锁版本号 |

## 三、周报核心查询关联关系

```text
issues (主表)
├── FK tracker_id    → trackers.id        (需求类型)
├── FK project_id    → projects.id        (所属项目)
├── FK status_id     → issue_statuses.id  (状态: 新建/进行中/已解决/已关闭/已拒绝/待验证)
├── FK priority_id   → enumerations.id    (优先级: P0~P3)
├── FK author_id     → users.id           (提单人)
├── FK assigned_to_id→ users.id           (负责人)
├── FK category_id   → issue_categories.id(分类)
├── FK fixed_version → versions.id        (目标版本)
├── FK parent_id     → issues.id          (父需求)
└── FK root_id       → issues.id          (根需求)

journals (变更日志)
├── FK journalized_id → issues.id
├── FK user_id        → users.id

time_entries (工时)
├── FK issue_id   → issues.id
├── FK project_id → projects.id
├── FK user_id    → users.id
```
