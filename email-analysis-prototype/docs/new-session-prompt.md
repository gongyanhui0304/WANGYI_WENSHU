# New Session Prompt For Business Users

Use this in a Codex session after IT has configured the `emailProjectAnalysis` MCP connection:

```text
请使用 emailProjectAnalysis 查询我有权限访问的邮件索引。先确认我能访问哪些邮箱，再根据我的问题查询摘要、线程和证据。不要读取本地邮件文件，也不要直接读取服务器路径。
```

Example questions:

```text
我能访问哪些邮箱？
这个邮箱最近主要在处理什么？
付款审批现在怎么样？
有哪些风险？
打开支持这个结论的邮件依据。
```
